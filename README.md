# Svelte Login Page

This repository contains a standalone Svelte login experience under `frontend/`. The demo focuses on a centered card, inline validation, and polished states.

## Getting started

```bash
cd frontend
npm install
npm run dev
```

## Features

- Responsive login card with username/email and password fields.
- Inline validation errors and disabled submit button until inputs are valid.
- Simulated submission state for a more realistic feel.
